## Prerequisites

- [TypeScript](https://www.typescriptlang.org/) version 4.6 or later.

- You must enable `strict` mode in your `tsconfig.json` file's `compilerOptions`.

```ts title="tsconfig.json"
{
  // ...
  "compilerOptions": {
    // ...
    "strict": true
    // ...
  }
  // ...
}
```
